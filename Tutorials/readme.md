# Pico Beginner Tutorials

-   Wire up the LED to a GPIO pin
    -   Write a program that blinks LED
    -   Wire up a button to a different GPIO pin
        - ~pull-up vs. pull-down resistor~
    -   Write a program that uses input from button to control LED


-   Reaction timer - Who has the fastest reaction time!
    -   Write  program that:
        -    Wait random delay
        -    LED turns on
        -    Press button as fast as possible
        -    Reaction time printed in console

-   Use PWM to controll the brightness of the LED
    -   Code button interrupt to change the brightness

-   Wire up photoresistor to an ADC pin
    -   Voltage divider!
    -   Write a program to read the photoresistor from the ADC pin (*hint: it reads 16 bits*)
